DOM4J简介

Dom4j是一个易用的、开源的库，用于XML，XPath和XSLT。它应用于Java平台，采用了Java集合框架并完全支持DOM，SAX和JAXP。

1.自己创建Document对象

2.通过读取XML文件获取Document对象

3.读取XML文本内容获取Document对象
